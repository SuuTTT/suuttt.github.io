Distill-style experiments on Hugo PaperMod

Distill posts are really just long-form HTML with inline CSS and a pinch of JavaScript for scrollytelling or interactive notes. You can recreate that feel on Hugo + PaperMod without any backend by leaning on three patterns, and each pattern maps neatly to common AI explainers.

Tiny interactive widgets via shortcodes

Wrap small, in-line widgets (temperature sliders, activation previews, etc.) in shortcodes so authors can drop {{< softmax-widget >}} inside Markdown. The shortcode template can live in layouts/shortcodes/softmax-widget.html and include both the markup and a minimal script:

<div class="distill-widget">
  <label>
    Temperature
    <input type="range" min="0.2" max="2.5" step="0.1" value="1" oninput="updateTemp(this.value)">
  </label>
  <div class="logits">logits: [1.2, 0.3, -0.1]</div>
  <div class="probs" aria-live="polite">softmax: [0.58, 0.27, 0.15]</div>
</div>
<script>
  const logits = [1.2, 0.3, -0.1];
  const softmax = (xs, t) => {
    const exps = xs.map(x => Math.exp(x / t));
    const sum = exps.reduce((a, b) => a + b, 0);
    return exps.map(x => (x / sum).toFixed(2));
  };
  function updateTemp(t) {
    const probs = softmax(logits, Number(t));
    document.querySelector('.probs').textContent = `softmax: [${probs.join(', ')}] (T=${t})`;
  }
</script>
<style>
.distill-widget { border:1px solid #ececec; padding:12px; border-radius:8px; display:flex; flex-direction:column; gap:0.5rem; }
.distill-widget label { font-weight:600; display:flex; justify-content:space-between; gap:0.75rem; align-items:center; }
.distill-widget input[type=range] { flex:1; }
.distill-widget .probs { font-family: var(--code-font, 'Source Code Pro', monospace); }
</style>

The shortcode is pure HTML/CSS/JS, so it survives GitHub Pages builds and keeps the Distill vibe.

Bigger experiments via iframes

For heavier sketches (attention heatmaps, 2D latent walks, WebGL demos), export them as standalone static HTML files and place them under static/experiments/. Then embed with an iframe from your post body:

<figure class="distill-embed">
  <iframe src="/experiments/attention-demo/index.html" title="Attention demo" loading="lazy"></iframe>
  <figcaption>Fully static embed that still works on GitHub Pages.</figcaption>
</figure>
<style>
.distill-embed iframe { width:100%; min-height:420px; border:1px solid #e0e0e0; border-radius:8px; }
.distill-embed figcaption { text-align:center; color:#555; margin-top:0.5rem; font-size:0.95rem; }
</style>

This keeps interactivity isolated and lets you iterate on the experiment without touching the main post content.

Custom layouts for special posts

When a piece needs a bespoke Distill-like layout (floating sidenotes, scrolling stepper, etc.), create a custom single layout: layouts/_default/single.distill.html. In your front matter, set layout: distill to opt in. The layout can add sticky sidenotes and include a dedicated stylesheet at assets/css/distill.css for typography tweaks.

For step-by-step AI explainers, a minimal scroll-step pattern keeps the HTML lightweight:

<section class="stepper">
  <div class="step" data-label="1">
    <h4>Layer norm</h4>
    <p>Normalize tokens, keep residual stream.</p>
  </div>
  <div class="step" data-label="2">
    <h4>Multi-head attention</h4>
    <p>Show per-head weights in an adjacent canvas.</p>
  </div>
  <div class="step" data-label="3">
    <h4>MLP</h4>
    <p>Switch activations (GELU/ReLU) with buttons.</p>
  </div>
</section>
<style>
.stepper { display:grid; gap:1rem; border-left:3px solid #111; padding-left:1rem; }
.step { position:relative; }
.step::before { content: attr(data-label); position:absolute; left:-2.6rem; top:0; font-weight:700; }
</style>

Author workflow tips

  • Keep reusable CSS (grid helpers, note callouts) in assets/css/distill.css and import it globally via params.assets in config.yaml.
  • Namespace your JavaScript (e.g., window.distillWidgets = {...}) to avoid conflicts with PaperMod scripts.
  • Document each shortcode with a minimal usage example so future posts stay consistent.
